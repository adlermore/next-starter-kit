## Features

- **Next.js 16** with App Router & TypeScript  
- **TailwindCSS 4** for utility-first styling  
- **Internationalization** with `next-intl`  
- **Form handling** with `react-hook-form` & `yup` validation  
- **Interactive UI**:  
  - Carousels with `embla-carousel-react`  
  - Animations with `framer-motion`  
  - Hamburger menus with `hamburger-react`  
  - Icon library: `lucide-react`  
- **Media support**: video playback via `react-player`  
- **Toast notifications** with `react-hot-toast`  
- **Sass support** for modular styling  
- **HTTP requests** via `axios`

## Tech Stack

**Frontend:**  
- React 19.x  
- Next.js 16.x  
- TypeScript 5.x  
- TailwindCSS 4.x + PostCSS  

**UI & Animations:**  
- Framer Motion  
- Embla Carousel React  
- Lucide Icons  
- Hamburger React  

**Forms & Validation:**  
- React Hook Form  
- Yup  
- @hookform/resolvers  

**Utilities:**  
- Axios for HTTP requests  
- clsx for conditional classNames  
- react-hot-toast for notifications  
- react-player for embedded media  
- @fancyapps/ui for fancy modals/lightboxes

- Node.js v20+

```bash
